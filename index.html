Personal Portfolio Tracker

跨券商、多币种的个人持仓管理工具。以“目标占比”为核心，自动统一到 JPY 口径，支持价格与汇率自动更新；数据仅保存在浏览器或仓库中。 建议先打开 Web App（英文按钮），或阅读方法论与 Quickstart（中文说明）。下方提供 Data Spec 与 Schemas（开发者参考）。

Open Web App Methodology (ZH) Quickstart (ZH) PRD v3 (ZH) Data Spec Google Sheets Guide (ZH) Google Sheet (legacy)
